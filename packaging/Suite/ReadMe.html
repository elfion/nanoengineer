The Nanorex NanoEngineer-1 1.1.1 Application Suite

NanoEngineer-1 (NE1) is an open-source (GPL) 3D multi-scale modeling and simulation program for nano-composites with special support for structural DNA nanotechnology. It features an easy-to-use interactive 3D graphical user interface for designing and modeling large, atomically precise composite nanosystems.

In addition to NE1, this Suite includes:

  • GROMACS - an extremely high performance, open source, and cross-platform package used to perform molecular dynamics simulation of systems with hundreds to millions of particles. NE1 currently uses GROMACS as an energy minimization engine option. 
  • QuteMolX - an interactive, high quality molecular visualization application aimed at improving clarity and understanding of the 3D shape and structure of large complex molecules. Users can "launch-and-load" QuteMolX directly from NE1. Learn more...
This is the NanoEngineer-1 1.1.1 Application Suite as of July 16, 2008.

System Requirements

Minimum

  • 2.5 GHz Intel Pentium 4 CPU or equivalent
  • Windows XP, Mac OS X 10.4, Linux
  • 1 Gb RAM
  • 350 Mb available disk space
  • OpenGL compatible 3D video card

Recommended

  • 3 GHz Intel Core 2 CPU or equivalent
  • Windows XP, Mac OS X 10.4, Linux
  • 2 Gb RAM
  • 350 Mb available disk space
  • OpenGL compatible 3D video card with FBO and VBO support (this web page gives a good list of recommended video cards.)

Installation Instructions

Windows

Run the NanoEngineer-1 Applications Suite self-extracting installer "NanoEngineer-1_Suite_1.1.1.exe".

Mac OS X

Run the NanoEngineer-1 Applications Suite self-extracting installer "NanoEngineer-1_Suite_1.1.1.dmg".

Linux and Developers

Each application in the Suite must be compiled from sources for Linux users and developers. Download the NanoEngineer-1 1.1.1 Application Suite source distribution "NanoEngineer-1_Suite_1.1.1.tar.gz" and follow the instructions found in each application's ReadMe file.
